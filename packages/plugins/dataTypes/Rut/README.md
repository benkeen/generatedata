# [Docs](../../../../../docs/README.md) &raquo; [Plugins](../../README.md) &raquo; [Data Types](../README.md) &raquo; Rut

This Data Type generates a random Chilean Unique National Role Number


### Example API Usage

```javascript
{
    "numRows": 20,
    "rows": [
        {
            "type": "Rut",
            "title": "Rut"
        }
    ],
    "export": {
        "type": "JSON"
    }
}
```
