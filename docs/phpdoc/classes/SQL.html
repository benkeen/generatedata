For Data Types to properly integrate with the SQL Export Type, they should return additional metadata that lets this module know about the database column type.

That value is used in the initial CREATE TABLE statement.

    "sqlField"          This is used for all SQL database types.
    "sqlField_Oracle"   Used for Oracle only
    "sqlField_MySQL"    Used for MySQL only
    "sqlField_SQLite"   Used for SQLite only
