    A generic class for handling all of the Core's Ajax requests.

    All requests are identified through a unique "action" string, and (usually) arbitrary other info passed via POST.

    author Ben Keen
    package Core

     Methods

    AjaxRequest objects are automatically processed when they are created, based on the unique $action value.

    __construct($action, $post) 

    The result of the call is stored in $response to be handled however you need (e.g. output as JSON, XML etc) - or an Exception is thrown if something went wrong. Exceptions are used SOLELY for program errors: not for user-entry errors.
